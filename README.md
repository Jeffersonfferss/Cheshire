# Cheshire
Cheshire es la llave a un jardín donde el tiempo te pertenece. Como el guía invisible de Alicia, la app aparece para facilitar cada movimiento y se desvanece para que nada interrumpa tu recorrido. Una herramienta sutil que transforma el campo en un mundo de maravillas, centrada en la fluidez, el detalle y el placer de juga
